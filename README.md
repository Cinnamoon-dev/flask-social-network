## Flask API
Using flask to create a minimal application of an API and SQLAlchemy as ORM for a SQLite database. 

### Database Configuration
#### Table Generation
> flask db init</br>
> flask db migrate</br>
> flask db upgrade</br>

### Running the Application
> python3 run.py </br>

### Creating and using the virtual environment
> poetry install </br>
> poetry shell </br>
